## How to sign

Signing is handled automatically: when you open a pull request that requires the
CLA, a bot will comment with instructions. Replying to that comment with the
exact sentence it asks for constitutes your electronic signature and acceptance
of this Agreement, recorded together with your GitHub username, the date, and
the pull request number.

You register your acceptance **once per version** of this Agreement — the bot
will not prompt you again while this version is current. As stated above, each
pull request you submit afterwards constitutes a renewed acceptance and a fresh
grant for that specific contribution; the registration only saves you from
retyping the sentence.
